mybatisPlus

mybatis快速入门

  1. 首先准备一张数据库表

  2. 创建一个空的springboot工程

  3. 添加依赖

    ```java
     <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter</artifactId>
        </dependency>
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-test</artifactId>
            <scope>test</scope>
        </dependency>
        <dependency>
            <groupId>com.baomidou</groupId>
            <artifactId>mybatis-plus-boot-starter</artifactId>
            <version>最新版本</version>
        </dependency>
        <dependency>
            <groupId>mysql</groupId>
            <artifactId>mysql-connector-java</artifactId>
        </dependency>     
    ```

    4.yml文件中配置相应的数据库相关的配置

    ```java
        //配置数据库
        spring:
      datasource:
        driver-class-name: com.mysql.jdbc.Driver
        username: root
        password: 1025
        //配置表头前缀
        mybatis-plus:
      global-config:
        db-config:
          table-prefix: ms_
    ```

5.在springboot的启动类中添加@MapperScan（包名）注解，扫描Mapper文件夹

6.编写实体类User.java,需要和表名一一对应，@TableName()来指定数据库表名，@TableId()用来指定主键

7.编写 Mapper 包下的 UserMapper接口，使其继承BaseMapper

8.编写测试代码即可

    ```java
    @SpringBootTest
    class BlogApplicationTests {
    @Autowired
    private adminMapper adminMapper;
        @Test
        void contextLoads() {
            System.out.println("---selectALL----");
           List<admin> adminList=adminMapper.selectList(null);
           adminList.forEach(System.out::println);
        }
    
    }
    ```

代码生成器

  • 设置依赖

    <dependency>
        <groupId>com.baomidou</groupId>
        <artifactId>mybatis-plus-generator</artifactId>
        <version>最新版本</version>
    </dependency>

  • 使用模板快速生成

     String url="jdbc:mysql:///blog?useSSL=false&serverTimezone=GMT%2B8";

                    .packageConfig(builder -> {
